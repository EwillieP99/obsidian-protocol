# Core Features & Mechanics

## The 12 Cyberpunk Block Types

**Structure**
- Obsidian, Chrome, Corp Glass

**Neon**
- Cyan Neon, Magenta Neon

**Energy**
- Toxic Core, Power Line

**Data**
- Data Stream, Holo Billboard, Circuit Plate, Neural Node

**Anomaly**
- Glitch Zone

Each block has unique stability, anomaly rating, emissive behavior, and (for 6 of them) custom animated GLSL shaders.

## Brush System
- 5 Modes: Paint, Purge (erase), Fill, Rewrite, Sample
- 3 Shapes: Cube, Sphere, Plane
- Adjustable size + randomness
- Live 3D preview with shape envelope

## Multi-Layer Editing
- 12 vertical layers
- Per-layer visibility, lock, solo, opacity, reordering
- Smart layer-aware brush application

## History & Undo/Redo
- Full chrono-log with visual timeline
- Click any entry to time-travel
- Cyan/magenta flash feedback on affected cells

## Corporate Contracts
- Procedurally generated high-stakes jobs
- Lore-rich starting structures
- Risk/reward scaling

## Persistence
- Autosave every 20 seconds (IndexedDB)
- Named save slots
- JSON import/export
- 5 example vault saves included

## Visual & Audio Polish
- Reactive post-processing (Bloom scales with neural integrity)
- Screen shake + bloom flash on large operations
- 360 capped particle system
- Synthesized cyberpunk audio (Web Audio API)
- Quality presets with auto-degrade
